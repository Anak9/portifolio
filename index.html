Trabalhei no projeto Preemie Test - Light Scan Skinage onde precisei, inicialmente, me comunicar e aprender com uma equipe de médicos e outros profissionais da saúde. Meu papel foi compreender suas demandas e traduzi-las em um programa de gerenciamento e análise de dados, que gerava novas informações a partir dos dados coletados durante a pesquisa de campo.


O banco de dados utilizado foi o Firebase Realtime Database, um banco NoSQL. Desenvolvi uma aplicação para conectar com esse banco utilizando NodeJS, que fazia tratamento dos dados e automatizava cálculos, produzindo novos dados para a equipe. A maior parte do trabalho foi no back-end, mas também atuei no Front-end para implementar uma simples interface utilizando Javascript e HTML.

Ferramentas usadas: JavaScript | NodeJS | banco NoSQL (firebase realtime database) Link do projeto: skinage.medicina.ufmg.br

No projeto Descobridores da Matemática, trabalhei como desenvolvedora Fullstack, sendo responsável pelo desenvolvimento e aprimoramento de um site interativo. Implementei funcionalidades como um sistema de chat, conexão com um banco de dados SQL e automatização de tarefas.


Inicialmente, meu foco foi atender à principal demanda da equipe: adicionar interatividade a um site estático. Para isso, utilizei JavaScript, HTML e CSS. Com o avanço do projeto e novas ideias surgindo, desenvolvi um chat para o website, o que exigiu a criação de um banco de dados SQL, inicialmente no mySQL e posteriormente adaptado para o PostgreSQL, e a estruturação do servidor, implementado inicialmente em Node.js. Além disso, implementei um sistema simplificado para que a equipe pudesse adicionar novos conteúdos ao site com mais facilidade. Esse trabalho foi supervisionado por um servidor do Instituto de Ciências Exatas da UFMG, pois o site seria hospedado no portal oficial da instituição.


Durante todo o projeto também atuei como monitora em outras áreas que permitiram um contato com uma equipe diversa. Em um ambiente de trabalho diferente do meu habitual, pude desenvolver novas habilidades organizacionais e de gestão, como comunicação, capacidade de mediação, liderança e gestão de grupo e habilidades interpessoais e emocionais como empatia e paciência, escuta ativa, flexibilidade e criatividade.

Ferramentas usadas: JavaScript | HTML | CSS | SQL (mySQL) | NodeJS Link do projeto: www.icex.ufmg.br/descobridores
